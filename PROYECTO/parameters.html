Parámetros

En ES5, se tienen varias maneras de manejar funciones que necesitan valores por defecto, argumentos indefinidos y parámetros nombrados. Con ES6, se puede lograr todo esto y más utilizando una sintaxis concisa.

Parámetros por defecto (Default Parameters)

          
            function addTwoNumbers(x, y) {
              x = x || 0;
              y = y || 0;
              return x + y;
            }

En ES6, se pueden simplemente dar valores por defecto para los parámetros de una función:

          
            function addTwoNumbers(x=0, y=0) {
              return x + y;
            }
          
        
          
            addTwoNumbers(2, 4); // 6
            addTwoNumbers(2); // 2
            addTwoNumbers(); // 0

Parámetros Rest

En ES5, se manejaba un número infinito de argumentos de la siguiente manera:

          
            function logArguments() {
              for (var i=0; i < arguments.length; i++) {
              console.log(arguments[i]);
              }
            }

Utilizando el operador rest, se puede pasar un número indefinido de argumentos.

          
            function logArguments(...args) {
              for (let arg of args) {
                console.log(arg);
              }
            }

Parámetros Nombrados (Named Parameters)

Uno de los patrones en ES5 para manejar parámetros nombrados era usar el patrón options object, adoptado de jQuery.

          
            function initializeCanvas(options) {
              var height = options.height || 600;
              var width  = options.width  || 400;
              var lineStroke = options.lineStroke || 'black';
            }

Se puede obtener la misma funcionalidad utilizando destructuración como un parámetro formal de una función:

          
            function initializeCanvas(
              { height=600, width=400, lineStroke='black'}) {
                // Use variables height, width, lineStroke here
            }

Si se quiere hacer el valor entero opcional, se puede destructurar un objeto vacío:

          
            function initializeCanvas(
              { height=600, width=400, lineStroke='black'} = {}) {
              // ...
              }
