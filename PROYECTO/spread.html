Operador Spread

En ES5, se podía encontrar el valor máximo de un arreglo utilizando el método apply a Math.max:

          
            Math.max.apply(null, [-1, 100, 9001, -32]); // 9001

En ES6, se puede usar el operador spread para pasar un arreglo de valores para ser utilizados como parámetros de una función:

          
            Math.max(...[-1, 100, 9001, -32]); // 9001

Se pueden concatenar literals de arreglo fácilmente con la siguiente sintaxis intuitiva:

          
            let cities = ['San Francisco', 'Los Angeles'];
            let places = ['Miami', ...cities, 'Chicago']; // ['Miami', 'San Francisco', 'Los Angeles', 'Chicago']
