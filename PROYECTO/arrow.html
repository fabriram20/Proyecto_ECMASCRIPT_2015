Arrow Functions (Funciones Flecha)

Muchas veces se tienen funciones anidadas en las que se quiere preservar el contexto de this de su alcance léxico. Por ejemplo:

          
            function Person(name) {
              this.name = name;
            }
            Person.prototype.prefixName = function (arr) {
              return arr.map(function (character) {
                return this.name + character; // Cannot read property 'name' of undefined
              });
            };

Una solución común a este problema es almacenar el contexto de this usando una variable:

          
            function Person(name) {
              this.name = name;
            }
            Person.prototype.prefixName = function (arr) {
            var that = this; // Store the context of this
              return arr.map(function (character) {
                return that.name + character;
              });
            };

También se puede pasar en el contexto apropiado de this:

          
            function Person(name) {
              this.name = name;
            }
            Person.prototype.prefixName = function (arr) {
              return arr.map(function (character) {
                return this.name + character;
              }, this);
            };

Así como enlazar el contexto:

          
            function Person(name) {
              this.name = name;
            }
            Person.prototype.prefixName = function (arr) {
              return arr.map(function (character) {
                return this.name + character;
              }.bind(this));
            };

Utilizando Arrow Functions, el valor léxico de this no se oculta y se puede reescribir lo anterior:

          
            function Person(name) {
              this.name = name;
            }
            Person.prototype.prefixName = function (arr) {
              return arr.map(character => this.name + character);
            };

Arrow Functions son también más concisas cuando se usan en expresiones de función que solo retornan un valor:

          
            var squares = arr.map(function (x) { return x * x }); // Function Expression
          
        
          
            const arr = [1, 2, 3, 4, 5];
            const squares = arr.map(x => x * x); // Arrow Function for terser implementation

Mejores prácticas

  • Usar Arrow Functions cuando se necesita preservar el valor léxico de this.
  • Usar Arrow Functions en lugar de expresiones de función cuando sea posible.
