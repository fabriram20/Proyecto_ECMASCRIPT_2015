Reemplazando IIFEs con Blocks

Un uso común de IIFEs (Immediately Invoked Function Expressions) es encapsular valores dentro de su alcance. En ES6, ahora se tiene la habilidad de crear alcances basados en bloques y por lo tanto no limitados solamente al alcance basado en función.

Utilizando IIFEs:

          
            (function () {
              var food = 'Meow Mix';
            }());
            console.log(food); // Reference Error

Utilizando Blocks ES6:

          
            {
              let food = 'Meow Mix';
            };
            console.log(food); // Reference Error
