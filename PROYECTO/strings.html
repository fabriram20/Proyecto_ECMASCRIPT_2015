Strings

Con ES6, la librería estándar ha crecido inmensamente. Junto a estos cambios hay nuevos métodos que pueden usarse en strings, como .includes() y .repeat().

.includes()

          
            var string = 'food';
            var substring = 'foo';
            console.log(string.indexOf(substring) > -1);

En lugar de comprobar que el valor de retorno sea mayor que -1 para denotar contención de string, se puede simplemente usar .includes(), el cual retorna un booleano:

          
            const string = 'food';
            const substring = 'foo';
            console.log(string.includes(substring)); // true

.repeat()

          
            function repeat(string, count) {
              var strings = [];
              while(strings.length < count) {
                strings.push(string);
              }
              return strings.join('');
            }

En ES6, ahora se tiene acceso a una implementacion más tersa:

          
            // String.repeat(numberOfRepetitions)
            'meow'.repeat(3); // 'meowmeowmeow'
